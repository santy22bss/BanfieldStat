Banfield Stats

Es una aplicación web integral diseñada para centralizar, visualizar y analizar el historial de partidos del Club Atlético Banfield en una interfaz moderna y eficiente.

* Contexto del Proyecto
Este software fue desarrollado como parte de mi formación. El objetivo principal es la exploración práctica de:

Integridad y Persistencia de Datos: Gestión de bases de datos locales mediante estructuras JSON para asegurar la consistencia histórica.

Seguridad en Entornos Web: Implementación de servidores locales seguros y manejo de rutas mediante Express.

Automatización de Procesos de Datos: Desarrollo de scripts para la extracción, sincronización y depuración de información deportiva.

Nota: Este es un proyecto de desarrollo personal con fines educativos y de optimización propia. Representa una fase de experimentación con flujos de trabajo AI-Assisted (Vibe Coding) utilizando el IDE Google Antigravity para acelerar el ciclo de vida del desarrollo.

* Características Principales
La aplicación se divide en tres pilares fundamentales:

- Visualización de Datos: Interfaz dinámica para la consulta de historial de partidos, equipos históricos (history.html).

- Gestión Inteligente de Datos: * Extracción: Scripts automatizados para la obtención de competencias y logos (extract_competitions.js, download_logo.js).

Debug: Mapeo y cálculo de errores en la base de datos para garantizar la precisión estadística.

- Automatización de Entorno: Centralización del despliegue mediante archivos de procesamiento por lotes para una inicialización inmediata del servidor local.

- Metodología: Vibe Coding & AI-Collaboration
A diferencia del desarrollo tradicional centrado en la sintaxis mecánica, este software fue construido mediante un flujo de iteración conversacional e ingeniería de prompts en Google Antigravity.

¿Qué significa esto en este proyecto?

Visión sobre Sintaxis: El enfoque principal fue la arquitectura de la base de datos y la lógica de negocio, delegando la implementación del boilerplate de Node.js y scripts de mapeo a la IA del entorno.

Refactorización Ágil: La lógica de promedios (debug_calc.js) y la interfaz de usuario fueron refinadas mediante ciclos rápidos de feedback humano-IA, permitiendo una evolución fluida del código.

* Detalles Técnicos
Lenguaje: JavaScript (ES6+)

Entorno de Ejecución: Node.js

Framework Web: Express

IDE: Google Antigravity (IA-Driven)

Persistencia: JSON (database.json, competitions.json)

* Instalación y Uso

Clona el repositorio:

Bash
git clone https://github.com/tu-usuario/paginaBanfield.git
Instala las dependencias:

Bash
npm install
Ejecución rápida: Inicia la aplicación ejecutando el archivo inicar.bat o mediante el comando:

Bash
node server.js
